## Prerrequisitos

1. Node js
    * https://nodejs.org/es/download/
2. Docker
    * https://docs.docker.com/get-docker/
3. Express
    * https://expressjs.com/es/

## Instalación

```bash
npm install
```

## Desarrollo

Ejecutar `npm run dev` para un servidor de desarrollo. Navega a `http://localhost:3001/`. La aplicación se recargará automáticamente si cambias cualquiera de los archivos fuente si estas en el modo watch.

```bash
$ npm run dev
```
